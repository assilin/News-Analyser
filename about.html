For clients

This website is a graduation project of a student from ValenciaJS. Its purpose is to showcase what the student has learned and the technologies they are proficient in.


The website's layout is built using the BEM methodology, making it flexible and scalable. Additionally, adding new pages and functionality is easier and faster compared to projects built using other methodologies. Moreover, the website is easier and more cost-effective to maintain.


The news cards are fetched from the server and processed asynchronously. This means that the website continues to function while waiting for a response from the server. This enhances the usability of the service.


Webpack was used as the build tool for this project. It is a modern approach to website development. The code is optimized, resulting in faster website performance. The website is also configured to work well in older browsers. Even if visitors are using outdated operating systems, they will not encounter any display issues. This expands the potential audience.


The project has been reviewed by experts from ValenciaJS. If your business is in need of a website, we recommend considering this student for the task.
